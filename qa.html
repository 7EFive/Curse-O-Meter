Q&A Section

What was the goal of the Project?

First of all - the goal was to pass this module.

Second of all - I wanted to try to make a gimic based website, since it's much more fun to interact with something, instead of just making a plane portfolio webpage

Last but not least, I always wanted to try making cool-looking UIb elements such as buttons with interesting designs, that convey some sort of emotion or different levels of comfort.

Which Programming languages were used for this website?

The website were written in HTML and formatted in CSS. Visualizing of the icons was made in Affity Designer. The random-generated facts part was written in JavaScript.

Why the cursed thematic?

At the time making this site, i was interested in some unusual facts, which were sometimes interesting and sometimes morbid, so i thought choosing that theme was a no-brainer.

Which part of the project did you enjoyed the most?

Experementing a bit with css,htaml and creating logos and/or icons for this page.

Which part of the project did you struggled the most?

Implementing and understanding the JavaScript part of the project, since i haven't done any JavaScript coding in years. I really had to look up some code online or ask other people to give me some advice on how to improve the code. But after finding some code lines that are familiar from other programming languages, it became a bit easier to implement and understanding the language.

Did you took anything away from this Project for yourselfe?

Of course! Not only did i refresh my knowledge of JavaScript, i also gained a better understanding on how to adapt a new code for personal gain, like in Java-Scripts case. Additionally, i found some new tools that are beneficial for illustrating on vector graphics, which will make creating images in some areas more convenient and versatile than before.
